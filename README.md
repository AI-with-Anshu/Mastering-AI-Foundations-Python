# Linear Regression Implementation from Scratch

This repository contains a fundamental implementation of a Linear Regression model using Python and Scikit-Learn. As an M.Tech student specializing in AI, I believe in mastering the core mathematical patterns that drive machine learning.

## 🚀 Project Overview
In this project, I have built a predictive model that identifies the relationship between input features ($X$) and target values ($y$). The model learns the pattern $y = 2x$ through supervised learning.

### Key Learnings:
- **Data Structuring:** Using NumPy to create 2D arrays (Matrices) for features and 1D vectors for targets.
- **Model Training:** Implementing `LinearRegression()` and using the `.fit()` method to optimize weights.
- **Prediction:** Using `.predict()` to estimate outcomes for unseen data points.
- **Visualization:** Plotting the regression line using Matplotlib to visualize how the AI fits the data.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** NumPy, Scikit-Learn, Matplotlib

## 📊 Visual Representation
The code includes a visualization script that generates a scatter plot of the actual data points and a regression line representing the AI's learned pattern.
